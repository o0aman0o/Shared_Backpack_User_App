## Installation

Shared Backpack system prerequisites
- [Flutter SDK][fsdk]
- [Android Studio][astudio]



Clone and change into the project
```sh
$ git clone https://github.com/Incentive-Coders/Shared_Backpack_User_App.git
```

Install packages
```sh
$ flutter pub get
```
